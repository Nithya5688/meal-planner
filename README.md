# Meal Planner

An Angular application that accepts ingredients and provides users with **high protein**, **high fiber**, and **balanced** meal recipes.

## Features

- **Ingredient input**: Add ingredients you have on hand (e.g., chicken, quinoa, spinach)
- **Recipe filtering**: Filter by High Protein, High Fiber, or Balanced meals
- **Recipe matching**: Recipes are matched based on your ingredients
- **Nutrition info**: Each recipe shows protein, fiber, and calories per serving
- **Expandable instructions**: Click to show/hide full cooking instructions

## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Installation

```bash
cd meal-planner
npm install
```

### Run the app

```bash
npm start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

### Build for production

```bash
npm run build
```

## Recipe Categories

- **High Protein**: Recipes with 25g+ protein per serving (e.g., grilled chicken, tuna, Greek yogurt)
- **High Fiber**: Recipes with 8g+ fiber per serving (e.g., lentils, black beans, oat bran)
- **Balanced**: Well-rounded macros for everyday eating (e.g., salmon bowls, stir-fried turkey)
